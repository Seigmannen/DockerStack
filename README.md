# Docker Stack

This is an example self-hosted Docker stack composed of multiple services grouped by category.  
Go through the stack and adjust configurations, paths, and settings to your liking before deploying.

## Core
- Portainer

## Databases
- MariaDB
- Redis

## Frontend
- Homer
- OpenWebUI
- phpMyAdmin
- Redis Commander

## Media
- Audiobookshelf
- FreshRSS
- PhotoPrism
- Plex Media Server

- qBittorrent
- Libation

## PVRs
- Lidarr
- Radarr
- Readarr for audiobooks
- Readarr for ebooks
- Sonarr
